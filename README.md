## 🌟 About The Project

This portfolio is designed to be a sleek, high-performance web application that leaves a lasting impression. It features a dynamic user interface with custom animations, custom scroll tracking, and environment-based backgrounds that react seamlessly to user preferences.

### ✨ Key Features

- **🎨 Dual Theme System:** 
  - **Dark Mode:** Immersive deep space galaxy background with slowly orbiting planets and twinkling stars.
  - **Light Mode:** Serene sky background with horizontally panning clouds, complete with parallax-scrolling birds and trees that emerge as you scroll down.
- **⚡ Lightning Fast:** Powered by Vite and React for instant load times and snappy page transitions.
- **🪄 Smooth Animations:** Utilizes Framer Motion for scroll-reveals, hover micro-interactions, and modal popups.
- **📱 Fully Responsive:** Carefully crafted with Tailwind CSS to look stunning on desktops, tablets, and mobile devices.
- **🧭 Scroll Tracking:** Features a dynamic left-sidebar timeline that highlights your current reading position.

## 🛠️ Tech Stack

- **Framework:** [React 19](https://react.dev/) + [Vite](https://vitejs.dev/)
- **Styling:** [Tailwind CSS v4](https://tailwindcss.com/)
- **Animations:** [Framer Motion](https://www.framer.com/motion/)
- **Icons:** [Lucide React](https://lucide.dev/)
- **Deployment:** [GitHub Pages](https://pages.github.com/) (Automated via `gh-pages` package)

## 🚀 How to Run Locally

To get a local copy up and running, follow these simple steps.

### Prerequisites
Make sure you have Node.js installed on your machine.
- npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. **Clone the repo**
   ```sh
   git clone https://github.com/hansiherath/portfolio.git
   ```
2. **Navigate into the directory**
   ```sh
   cd portfolio
   ```
3. **Install NPM packages**
   ```sh
   npm install
   ```
4. **Start the development server**
   ```sh
   npm run dev
   ```
   *The application will be available at `http://localhost:5173/`*

## 🌐 Deployment

This project is configured for automated deployment to GitHub Pages.

To build the production files and push them live, simply run:
```sh
npm run deploy
```
*Note: Make sure your code is committed and pushed to your GitHub repository before deploying!*
